Etapas del Proyecto

1. Armado del dataset

En esta etapa se definieron los objetivos del análisis y se recopiló la información proveniente de distintas fuentes. Se normalizaron los datos y se organizaron en un formato estructurado que facilitara su uso posterior en el proceso de análisis.

2. ETL con Power Query

Se utilizó Power Query para realizar la transformación de los datos: limpieza, combinación de tablas, eliminación de duplicados y creación de columnas calculadas. Esta fase permitió automatizar el flujo de carga de datos y mejorar la calidad del dataset.

3. Modelado con Power Pivot

Se construyó el modelo de datos utilizando Power Pivot, relacionando las tablas con claves primarias y foráneas. Se crearon medidas con DAX y se optimizó el modelo para lograr un análisis eficiente en los reportes dinámicos.

4. Visualización y análisis

Se diseñaron visualizaciones interactivas con tablas dinámicas, gráficos y segmentaciones. Se realizó un análisis exploratorio y se destacaron los hallazgos más relevantes, aportando valor a la toma de decisiones mediante un tablero visual claro y funcional.
